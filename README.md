## Sobre mí

- **Desarrollador Full Stack** con base en Python: construyo aplicaciones completas con Django, Flask, PostgreSQL y MySQL, desde el modelo de datos hasta la interfaz.
- **Superviso el área de TI** de un hospital en Bogotá, garantizando la continuidad de servicios críticos 24/7.
- He desarrollado desde **sistemas de gestión empresarial multi-tenant** hasta una **plataforma fintech de créditos digitales** — el dominio cambia, la calidad no.
- Mi especialidad: unir **diseño limpio** con **arquitecturas backend robustas y escalables**.
- Actualmente cursando **Ingeniería de Software** (Universidad Iberoamericana).
